BreakDown Sheet Generator

This module in the C3 Excel Tool allows users to generate or update a comprehensive "BreakDown" sheet by merging data from two input sources:

  • QS: Quantity Survey data containing Cost Account, BOQ Item, Budget Qty, UOM, and Template Code.
  • Database_Temp: A resource database containing resource details associated with each Template Code.

Key Features

  • Automatically checks for existing "BreakDown" sheet and updates only new cost accounts.
  • Creates and formats the "BreakDown" sheet if it does not exist.
  • Merges data from QS and Database_Temp based on Template Code.
  • Auto-fits columns for clear visibility.
  • Displays a success message indicating whether it was a creation or update.

Input Sheets

Sheet Name Required Columns
QS Cost Account, BOQ Item, Budget Qty, UOM, Template Code
Database_Temp Template Code, Template Name, Resource Code, Resource Name, RE.U.O.M, Productivity Ref, No. Of Labors, Remarks, Driving Resource, Activity ID

How to Use

  1. Ensure both QS and Database_Temp sheets exist and are filled.
  2. From the ribbon, click the “Generate/Update BreakDown” button.
  3. The tool will:
    • Check for existing BreakDown sheet or create it.
    • Read and merge data from both sources.
    • Append only new Cost Accounts.
    • Auto-format the result.
  4. A message will confirm whether the sheet was created or updated.

Important Notes

  • If the BreakDown sheet exists, only new Cost Accounts from the QS sheet will be appended.
  • Rows with duplicate Cost Accounts are skipped.
  • The tool uses a hidden in-memory collection to track existing entries and prevent duplicates.
💡 Tip: Before running this tool, ensure that the QS and Database_Temp sheets are up to date and that Cost Account and Template Code values are filled properly. Always create a copy of your file before running the tool, as changes are irreversible and Excel has no Undo for macros.
