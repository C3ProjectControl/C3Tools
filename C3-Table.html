Overview

The C3 Table Format module allows users to convert selected Excel ranges into styled tables that conform to C3's visual standards. This enhances readability, ensures uniform reporting, and supports Excel features like sorting, filtering, and total rows.

Key Features

  • Converts any range into an Excel table with automatic styling
  • Prompts user for header detection and applies conditional formatting
  • Alternating row colors for better data visualization
  • Supports existing tables: apply formatting without recreating

How to Use

  1. Select a cell or a range in your worksheet
  2. Click the C3 Table Format button in the ribbon
  3. If the range is not already part of a table, a prompt will ask if the selection includes headers
  4. The tool will format the range using C3 standards or apply custom formatting if already a table

Prompts and Behavior

  • Header prompt: Confirms whether the first row includes headers
  • Existing table: If selected range is inside a table, the tool offers to reformat it
  • Table naming: A unique name is automatically assigned (e.g., Table_Sheet1_1)

Limitations & Tips

  • Formulas and merged cells are supported but should be reviewed post-formatting
  • Does not undo‚Äîmake a backup or duplicate sheet before applying
  • Only applies custom formatting to one table at a time
