Import XER

The Import XER feature allows users to directly import tables from a Primavera P6 XER file into Excel. It parses the file, lists the available tables, and gives the user control to select only the desired ones. This makes project data easily accessible for further analysis and manipulation.

Key Features

  • Supports direct import from .xer files generated by Primavera P6.
  • Auto-detects and lists all available tables inside the file.
  • Allows selective import via a user-friendly form interface.
  • Creates a new sheet for each selected table, named appropriately.
  • Auto-formats date fields with standard date-time formatting.
  • Utilizes a non-blocking progress bar for smoother experience during large imports.

How to Use

  1. Click the Import XER button on the ribbon.
  2. Select a .xer file exported from Primavera P6.
  3. A list of available tables will appear in a form â€” check the tables you wish to import.
  4. Click Import. A progress bar will show the status while data is processed.
  5. Each selected table will be added to a new worksheet in the current workbook.
