Gantt Chart Generator

The Gantt Chart button dynamically creates a weekly Gantt chart from user-specified start and finish dates. It converts raw task data into a visually rich, scrollable timeline with WBS-style formatting, progress tracking, and optional current day highlighting.

Key Features

  • Fully dynamic timeline that adjusts to your data‚Äôs date range.
  • Configurable start/finish/progress columns and header row via a selection form.
  • Milestones (1-day tasks) shown as markers using Wingdings symbols.
  • Progress shading is optional and only applied if a numeric progress column is provided.
  • WBS rows with custom fill colors automatically apply those colors to their corresponding Gantt bars.
  • Weekly headers with merged Year, Month, and Week Start labels.
  • Optional highlight for the current date using gold borders.
  • Auto-inserts separator rows between tasks for better readability.

How to Use

  1. Ensure your Excel sheet contains task data with columns like Activity, Start, Finish, and optionally Progress %.
  2. Click the Gantt Chart button from the ribbon.
  3. A setup form will appear prompting for:
    • Header Row ‚Äî select using the row selector button.
    • Activity, Start, Finish, Progress Columns ‚Äî each has a Select button to pick directly from the sheet.
    • Highlight Today ‚Äî toggle Yes/No.
  4. The tool analyzes the date range and generates the chart next to your existing data.
  5. No specific cell like A1 is required ‚Äî just ensure your columns and header row are clearly specified via the form.
