Overview

The Text Case Conversion module allows users to transform cell content casing quickly using a user-friendly Excel ribbon interface. This is ideal for cleaning up data entries, standardizing names, and ensuring consistency in reports.

Available Case Options

Button Description
Smart Proper Case Capitalizes each word while preserving acronyms and formatting punctuation like apostrophes or hyphens.
Classic Proper Case Uses Excelâ€™s built-in PROPER() function; capitalizes first letter of every word, but changes acronyms.
UPPER CASE Converts all text to uppercase.
lower case Converts all text to lowercase.

How to Use

  1. Open your Excel workbook.
  2. Select a range containing plain text (non-formula cells).
  3. Click the appropriate button from the ribbon (e.g., Smart Proper Case).
  4. Confirm the range when prompted.
  5. The selected text will be converted accordingly.
