Tabular View

The Tabular View button transforms a Primavera P6 WBS-exported table into a structured, readable tabular format in Excel. P6 WBS hierarchies are usually represented through indentation (leading spaces) in the Activity ID or WBS ID column. This tool parses that hierarchy and generates one column per level for easier analysis.

Key Features

  • Auto-detects WBS levels from indentation and separates them into WBS Level 0, Level 1, etc.
  • Extracts Activity ID and Activity Name explicitly.
  • Includes all remaining columns such as Start, Finish, Budgeted Units, etc.
  • Outputs the result in a new sheet titled Tabular View with auto-formatting applied.

How to Use

  1. In Primavera P6, copy the WBS table (ensure it includes WBS ID, Activity Name, and other relevant fields).
  2. Paste the data into Excel so that the header row starts at cell A1.
  3. Click the Tabular View button from the ribbon.
  4. The tool will analyze the indentation in the WBS ID/Activity ID to build a hierarchical view.
  5. A new sheet named Tabular View will be created with the formatted output.

⚠️ Important Notes

  • P6 data must begin at cell A1 with the header row in row 1.
  • The tool looks for column headers "Activity ID" and "Activity Name" — spelling must match (case-insensitive).
  • Rows without an Activity Name are treated as WBS headers (used to build the hierarchy).
  • Indentation is calculated from leading spaces in the WBS ID (2 spaces = 1 level).

Output Format

WBS Level 0 WBS Level 1 WBS Level 2 Activity ID Activity Name Start Finish ...
Project A Structure Foundations A1234 Excavation 01-May-2025 07-May-2025 ...
