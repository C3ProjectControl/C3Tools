WBS Grouping

The WBS Grouping tool is designed to structure Excel data that has been copied directly from Primavera P6. When you copy a WBS table from Primavera and paste it into Excel, it retains indentation using leading spaces in column A to represent the WBS hierarchy.

This button analyzes the number of leading spaces in each row and applies Excelâ€™s outline grouping accordingly, enabling collapsible levels that match the original structure from Primavera.

Key Features

  • Automatically detects WBS hierarchy by counting leading spaces in column A.
  • Groups rows based on structure copied from Primavera P6 WBS tables.
  • Supports up to 7 levels of nesting .
  • Clears any existing grouping before applying new outline levels.

How to Use

  1. In Primavera P6, open the WBS view or Activity table where the WBS structure is visible.
  2. Select the WBS section (usually the first column), copy it, and paste it into an Excel worksheet â€” this will preserve indentation via leading spaces.
  3. Make sure column A in Excel contains this pasted data with visible indents.
  4. Go to the sheet in Excel and click the WBS Grouping button from the ribbon.
  5. The button will automatically apply grouping based on indentation levels.

Example Input (from Primavera)

Column A (WBS/Activity Name) Detected Level
Project0
  Site Works1
    Excavation2
    Backfilling2
  Structure1
    Foundation2
    Columns2

The above structure will be transformed into Excel outline groups, making it easier to collapse or expand WBS levels.

Notes and Tips

  • This tool only analyzes text in Column A.
  • Make sure the pasted WBS from Primavera uses consistent spacing (usually 2 spaces per level).
  • Existing row grouping must be cleared manually before using this tool. The button does not reliably remove all previous outlines.
  • To clear outline: Go to Data â†’ Ungroup â†’ Clear Outline before running the button.
  • Maximum depth supported is 14 spaces (7 levels, assuming 2 spaces per level).
